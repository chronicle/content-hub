# CLI Tool (`mp`)

The `mp` (Marketplace) tool is the official command-line interface for developing, maintaining, and deploying Google SecOps Content-Hub content.

For full documentation, command references, and installation guides, please see the [**mp Package Documentation**](/packages/mp/README.md).
